# ls-openapi( 구 이베스트 증권)
ls 증권 open api 를 사용하기 위한 모듈

한국투자증권에서 제공되지 않는 정보를 제공한다. 


## rest api 를 제공하는 증권사
- 한국 투자 증권
- 이베스트 투자증권

## 1분봉 과거 데이터를 제공하는 증권사
- 키움증권 
  - C# 으로 구현 해야함

# 개발환경
- open jdk 17
