# stock-ds
- Relational Database Service
- Redis
- 기타 빅데이터 처리 시스템

위 서비스들은 사용자 단에서 직접 접근하는 경우는 없습니다. 직접 접근이 가능한 관련도가 깊은 개발자들이 사용하는 프로젝트이며 분석에 필요한 표준화된 데이터 구조로 변환시키는 작업을 합니다
